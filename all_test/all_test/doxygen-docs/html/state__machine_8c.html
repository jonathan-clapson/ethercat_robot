int state_machine ( )

State machine

The state machine currently only runs the wago stepper motors, it appears to run correctly in TwinCAT3 It definately runs correctly in soem. It would probably be a nice solution to create a queue on which to register these instructions instead. This would significantly simplify the code. Currently debug messages are printed whenever the state changes

Parameters
[in]m_TraceA reference to TwinCAT3's m_Trace object, needed for printf() in TwinCAT3. This argument is not present under SOEM.
Returns
Returns ERR_SUCCESS if the state machine is still executing and ERR_STATE_MACHINE_STOPPED once execution is complete.
